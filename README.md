# Project-2-Hybrid-Cloud-Migration-Dynamic-Pipelines
Migrated  enterprise-level On-Premise SQL data to Azure Cloud utilizing  Self-Hosted Integration Runtime (SHIR) . Authored dynamic pipelines in  ADF  using Lookup and ForEach activities to automate multi table ingestion workflows. Deployed watermark-based  incremental loading  and integrated  Azure Logic Apps  for automated failure alerting.
